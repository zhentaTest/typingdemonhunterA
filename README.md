# 타이핑 연습 - 프로그래밍 언어 튜터

프로그래밍 언어 코드를 타이핑하면서 연습할 수 있는 웹 기반 타이핑 튜터 서비스입니다.

## 기능

### 핵심 기능
- ✅ **실시간 타이핑 연습**: 상단에 원본 코드, 하단에 사용자 입력 표시
- ✅ **즉각적인 오류 표시**: 틀린 글자는 양쪽 영역 모두 빨간색으로 표시
- ✅ **동적 타이머**: 경과 시간을 초/분/시간 형식으로 자동 전환
  - 0-59초: "35초"
  - 1-59분: "01분 42초"
  - 1시간 이상: "01시간 15분 30초"
- ✅ **실시간 WPM 계산**: 분당 타수 실시간 업데이트
- ✅ **3가지 프로그래밍 언어**: TypeScript, JavaScript, GodotScript
- ✅ **3가지 난이도**: 초보, 중급, 달인
- ✅ **백스페이스 지원**: 잘못 입력한 글자 수정 가능
- ✅ **반응형 디자인**: 모바일 및 데스크톱 지원

### 기술 스택
- HTML5
- CSS3 (순수 CSS, Flexbox, Grid)
- JavaScript (ES6+, 순수 자바스크립트)

## 사용 방법

1. **시작하기**: 웹 브라우저에서 `index.html` 파일을 엽니다.

2. **언어 및 난이도 선택**:
   - 상단의 드롭다운 메뉴에서 원하는 언어를 선택합니다
   - 난이도를 선택합니다 (초보/중급/달인)

3. **타이핑 시작**:
   - 화면을 클릭하여 입력 활성화
   - 첫 글자를 입력하면 타이머가 자동으로 시작됩니다
   - 상단의 원본 텍스트를 보면서 하단에 타이핑합니다

4. **오류 수정**:
   - 백스페이스 키로 잘못 입력한 글자를 삭제할 수 있습니다
   - 틀린 글자는 빨간색으로 표시됩니다

5. **새로 시작**:
   - "새로 시작" 버튼을 클릭하여 다른 샘플로 연습할 수 있습니다

## 코드 샘플

각 언어별, 난이도별로 5개씩 총 45개의 코드 샘플이 제공됩니다:

### TypeScript
- **초보** (3-5줄): 변수 선언, 간단한 함수, 인터페이스
- **중급** (7-12줄): 클래스, 배열 메서드, 제네릭
- **달인** (15-25줄): 비동기 처리, 고급 타입, 추상 클래스

### JavaScript
- **초보** (3-5줄): 기본 문법, 변수, 함수
- **중급** (7-12줄): 클래스, 배열 조작, 콜백
- **달인** (15-25줄): async/await, 이벤트 시스템, 함수형 프로그래밍

### GodotScript
- **초보** (3-5줄): 변수, 상수, 간단한 함수
- **중급** (7-12줄): 노드 확장, 시그널, 열거형
- **달인** (15-25줄): 물리 처리, 상태 머신, 파일 입출력

## WPM 계산 방식

WPM (Words Per Minute) = (정확하게 입력한 글자 수 / 5) / 경과 시간(분)

- 표준 영어 타이핑 기준 (1 단어 = 5글자)
- 틀린 글자는 정확한 글자 수에서 제외
- 실시간으로 업데이트

## 개발 참고사항

### 주요 구현 사항

1. **타이머 시스템** (`app.js`):
   - `startTimer()`: 첫 입력 시 자동 시작
   - `updateTimer()`: 동적 형식 변환
   - 100ms마다 업데이트

2. **타이핑 로직** (`app.js`):
   - `handleInput()`: 실시간 입력 처리
   - `renderOriginalText()`: 원본 텍스트 렌더링
   - `renderUserInput()`: 사용자 입력 렌더링
   - 글자 단위 비교 및 오류 표시

3. **샘플 선택** (`app.js`):
   - 무작위 선택
   - 연속 중복 방지 로직
   - 언어/난이도별 필터링

4. **스타일링** (`style.css`):
   - Monospace 폰트 (Consolas, Monaco, Courier New)
   - 반응형 디자인 (768px, 480px 브레이크포인트)
   - 그라디언트 배경
   - 애니메이션 효과
